# Introduction :

in this module we are going to talk about casting, what is casting and what are the types of casting.

### what is casting 

casting is forcing a datatype to be converted to an other datatype.

in c++ there are 4 types of casting.

- static cast

- dynamic cast

- const cast

- reinterpret cast
